Klienttekniker

HTML

HTML är en förkortning för HyperText Markup Language och används för att ge struktur åt en hemsida i form av element. Dessa element består av bland annat rubriker, länkar och paragrager, i princip allt som en hemsida behöver kan byggas upp med HTML. Det går även att lägga till andra filer så som bilder eller annan media. HTML har utvecklats med tiden och olika versioner fills tillgängliga, idag är det HTML5 som är standard.

CSS

CSS står för "Cascading Style Sheets" och används för att positionera och ändra utseendet på en hemsida som är byggt på HTML element. CSS är en av grundpelarna till dagens internet tillsammans med HTML och JavaScript. En CSS fil kan ses som en mall som webläsaren tolkar för att kunna ge hemsidan rätt utseende CSS filen kan länkas sedan till en HTML fil för att ge strukturen som har skapats i HTML ett mer nyansrikt utseende.

JavaScript

JavaScript är det populäraste webprogrammeringspråket idag. Det är ett högnivå språk och är objektorienterat. Det används för att ge en statisk hemsida som är byggd med HTML och CSS fler funktioner som mer avancerade animationer och presentationer. JavaScript kan även användas för back-end programmering.

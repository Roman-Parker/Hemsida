Internet

Klient-server-modellen

Många av dagens webtjänster och andra applikationer använder sig av klient-server modellen som bygger på att en användare kör ett program lokalt på datorn eller annan enhet som kör ett program som till exempel en webbläsare som tjänar som klient. Klientens uppgift är att integera med användaren och ta emot information. Informationen som klienten tar emot skickas sedan till en server där informationen sparas och bearbetas.

Internetkommunikation

Internetkommunikationen sker genom ett stort nätverk av servrar, routrar och enheter så som mobiler och datorer som komunicerar med varandra antingen via fysiska anslutningar eller via trådlösa standarder som wifi och mobildata nätverk. Videon nedanför förklarar hur dagens internet fungerar lite mer ingående.


Protokoll

För att alla olika enheter och programvaror ska kunna kommunicera med varandra utan att det sker missförstånd eller att data skickas på fel sätt så har man skapat olika protokoller. Varje protokoll beskriver hur data ska skickas och hur den ska formatteras. Protokoll är lite som ett språk för datorer som alla kan förstå oberoende vad för hårdvara eller mjukvara de två enheterna använder. Kan jämnföras med engelskan som gör att vi människor kan komunicera med varandra även om vi har olika modersmål.

HTTP/HTTPS

HTTP/S är ett av de absolut vanligaste protokollen i dagens internet. HTTP står för "Hyper Text Transfer Protocol". S:et i HTTPS står för secure och har lagts till då några av funktioner av ett protokol som heter TLS har lagts till HTTP som gjort det möjligt att kryptera informationen och skapa ett säkrare protokoll. HTTPS är viktigt framförallt på öppna och osäkra nätverk det går att avlyssna trafiken om det föråldrade HTTP protokollet används.

FTP/FTPS

För överföring av filer över nätvärk används ofta program som implementerar FTP/FTPS protokollen. FTP är en förkortning för "File Transfer Protocol". Skillnaeden mellan FTP och FTPS är att FTPS har likt som HTTPS implementerat andra protokol som TLS/SSL vilket skyddar mot man-in-the-middle attacker som kan avlyssna trafiken och snappa up filer som skickas via FTP, detta är inte möjligt genom FTPS.
